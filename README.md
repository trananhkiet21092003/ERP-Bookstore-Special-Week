# ERP Bookstore Management System

## Project Overview
This project simulates an ERP system used to manage bookstore operations such as sales, inventory, and order management.

The goal of the project is to practice Business Analysis activities including requirement analysis, process modeling, and database design.

## Project Structure

### 1. ERP Bookstore Project Document
Contains business analysis artifacts:
- Project Definition
- Business Problem
- Business Process (AS-IS / TO-BE)
- Use Case List
- Database Design
- SQL Test Queries

### 2. BPMN & UML
Contains process diagrams and system modeling:
- BPMN diagrams
- UML diagrams

### 3. SQL
Contains database scripts:
- Table creation
- Sample queries
- Data validation queries

## Tools Used
- Draw.io (BPMN & UML)
- Excel (Use Case List)
- SQL
- GitHub (Project Repository)
